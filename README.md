# PAF-skill_hive

PAF-skill_hive
About
PAF-skill_hive is a project developed as part of the IT3030 - Programming Applications and Frameworks (PAF) module at SLIIT. This skill-sharing and learning platform enables users to create and share skill-based posts, track learning progress, and manage user profiles. Built using Spring Boot for the REST API and React with Tailwind CSS for the frontend, it supports features like skill-sharing, progress updates, and user interactivity.
Features

User authentication with OAuth 2.0
Skill-sharing posts and learning progress updates
User profile management
CRUD operations for skills and progress
Real-time notifications

Technologies Used

Frontend: React, Tailwind CSS
Backend: Spring Boot, MySQL
Authentication: OAuth 2.0
Version Control: Git

Installation

Clone the repository: git clone https://github.com/VenushanT/PAF-skill_hive.git
Navigate to the project directory: cd PAF-skill_hive
Install backend dependencies: cd server && mvn install
Install frontend dependencies: cd client && npm install
Set up the MySQL database and update application.properties with your credentials.
Start the backend: mvn spring-boot:run in the server folder.
Start the frontend: npm start in the client folder.

Usage

Access the application at http://localhost:3000.
Log in with OAuth 2.0 credentials.
Explore features like creating posts, updating progress, and managing profiles.
